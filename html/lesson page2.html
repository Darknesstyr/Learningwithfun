Lesson 2: COMPUTER
PARTS OF A COMPUTER:

1. What is a Computer?
A computer is an electronic device that processes data and performs tasks based on instructions provided by software. It comprises hardware (physical components) and software (programs and operating systems).



2. Input and Output Devices
Input Devices: Tools for providing data to a computer (e.g., keyboard, mouse, scanner, microphone).
Output Devices: Tools for receiving data from a computer (e.g., monitor, printer, speakers).



3. Hardware Components
CPU (Central Processing Unit): The brain of the computer that processes instructions.
RAM (Random Access Memory): Temporary memory used for running applications.
Storage: Long-term data storage (e.g., SSDs, HDDs).
Motherboard: The main circuit board connecting all components.
Power Supply: Converts electricity to power the components.



4. Software
Operating System (OS): Software that manages hardware and software resources (e.g., Windows, macOS, Linux).
Applications: Programs designed for specific tasks (e.g., word processors, browsers, games).



5. Network and Connectivity
LAN (Local Area Network): Connects devices within a local area like a home or office.
WAN (Wide Area Network): Connects devices over a larger geographical area (e.g., the internet).
Wi-Fi and Ethernet: Common methods for connecting computers to networks.



6. Storage Types
Volatile Storage: Data is lost when power is off (e.g., RAM).
Non-Volatile Storage: Data is retained even without power (e.g., SSDs, HDDs, USB drives).



7. File Systems and Management
Computers organize data using file systems:
File Types: Documents (.docx, .pdf), images (.jpg, .png), videos (.mp4), and executables (.exe).
Folders and Directories: Structures to keep files organized.



8. Security
Antivirus Software: Protects against malware.
Firewalls: Prevent unauthorized access to networks.
Strong Passwords: Essential for safeguarding accounts.
Data Backups: Ensure critical data is saved in case of system failure.



9. Common Computer Troubleshooting
Slow Performance: Check for unnecessary programs or malware.
No Internet: Verify cables, restart routers, or check network settings.
Freezing: Restart the computer or update the software.



10. Evolution of Computers
Generations: From early mechanical calculators to modern quantum computers.
Emerging Technologies: AI, machine learning, and cloud computing are shaping the future of computers.




HARDWARE:

1. What is Computer Hardware?
Hardware refers to the physical components of a computer system that you can touch and see. It includes input/output devices, processing units, and storage systems.



2. Key Hardware Components
CPU (Central Processing Unit): The "brain" of the computer. Executes instructions from software programs. Measured by clock speed (GHz) and number of cores.

Motherboard: The main circuit board connecting all components. Houses the CPU, RAM, and other connectors.

RAM (Random Access Memory): Temporary memory used for running programs and tasks. Measured in gigabytes (GB) and speed (MHz).

Storage Devices: HDD (Hard Disk Drive): Uses spinning disks to store data. SSD (Solid-State Drive): Faster and more durable than HDDs.



3. Input Devices
Devices that allow users to interact with the computer:
Keyboard: For typing and issuing commands.
Mouse: For navigating and interacting with graphical interfaces.
Touchscreens: Common in tablets and smartphones.



4. Output Devices
Devices that present information to the user:
Monitor: Displays visuals and graphical interfaces.
Printer: Outputs hard copies of digital files.
Speakers: Produces audio output.



5. Power Supply Unit (PSU)
Converts electricity from the wall socket into usable power for the computer's components.
Rated in watts (W) to match system power requirements.



6. Cooling Systems
Fans: Basic air cooling for components.
Heat Sinks: Dissipates heat from components like the CPU.
Liquid Cooling: Advanced cooling for high-performance systems.



7. Graphics Processing Unit (GPU)
Specialized hardware for rendering images, videos, and animations.
Essential for gaming, video editing, and AI applications.



8. Networking Hardware
Network Interface Card (NIC): Connects computers to networks.
Wi-Fi Adapter: Enables wireless network connectivity.
Routers and Modems: Essential for internet access.



9. Peripheral Devices
External Storage: USB drives, external SSDs/HDDs for extra storage.
Cameras: For video conferencing or capturing media.
Docking Stations: Expands connectivity for laptops.


10. Maintenance and Upgrades
Cleaning: Regularly clean internal components to prevent dust build-up.
Upgrades: Increase RAM, storage, or replace the GPU for better performance.
Diagnostics Tools: Use hardware monitoring software to check system health.




SOFTWARE:

1. What is Computer Software?
Software refers to the set of instructions, programs, or data used by a computer to perform tasks. Unlike hardware, software is intangible and operates the hardware to execute specific functions.



2. Types of Software
- System Software Manages hardware and provides a platform for running other software.

Examples:
* Operating Systems (OS) like Windows, macOS, Linux.
* Utility programs like antivirus, disk cleanup, and file management tools.

- Application Software Designed for end-users to perform specific tasks.

Examples:
* Productivity tools like Microsoft Office, Google Workspace.
* Entertainment apps like Spotify, Netflix.
* Web browsers like Chrome, Firefox.



3. Operating Systems (OS)
Acts as the intermediary between hardware and software.

Examples of OS:
- Windows: Popular for personal and business use.
- macOS: Optimized for Apple devices.
- Linux: Open-source, highly customizable.



4. Software Installation
Software can be installed from various sources:
- Physical media (CD/DVD).
- Download from official websites or app stores.
- Using package managers in Linux environments.



5. Programming Languages
Software is created using programming languages. Examples:
- High-Level Languages: Python, Java, C++.
- Low-Level Languages: Assembly, Machine Code.
- Scripting Languages: JavaScript, PHP, Bash.



6. Licensing and Distribution
Types of software licenses:
- Freeware: Free to use, e.g., VLC Media Player.
- Open Source: Free and modifiable, e.g., Linux.
- Proprietary: Requires purchase or subscription, e.g., Adobe Photoshop.
- Trialware: Free for a limited time, then requires payment.



7. Software Development Lifecycle (SDLC)
Stages in software development:
- Requirement Analysis: Understanding user needs.
- Design: Creating architecture and interface.
- Development: Writing and testing code.
- Deployment: Distributing the software to users.
- Maintenance: Updating and fixing issues.



8. Software Updates
Importance:
- Fixes bugs and vulnerabilities.
- Adds new features and improvements.

Updates can be:
- Automatic: System downloads and installs updates.
- Manual: User chooses to update.



9. Security Software
Protects systems from threats like viruses, malware, and hacking attempts.

Examples:
- Antivirus Programs: Norton, McAfee.
- Firewalls: Blocks unauthorized access to networks.
- Encryption Tools: Protect sensitive data.



10. Cloud Software
Software delivered over the internet instead of being installed locally.

Examples:
- Cloud storage: Google Drive, Dropbox.
- SaaS (Software as a Service): Canva, Salesforce.
- Benefits: Accessibility, automatic updates, cost-effectiveness.
